Intermediate JavaScript,API Related Question & Answer

Question-1:Discuss the scope of var, let, and const.

Answer:var declarations are globally scoped or function scoped while
let and const are block scoped. var variables can be updated and re-
declared within its scope; let variables can be updated but not re-
declared; const variables can neither be updated nor re-declared.

Question-2:Tell us the use cases of null and undefined.

Answer:Undefined means the variable has been declared, but its value
has not been assigned. Null means an empty value
or a blank value. The typeof() operator returns undefined
for an undefined variable. The typeof() operator returns
type as an object for a variable whose value is assigned as null.

Question-3:What do you mean by REST API?

Answer:A REST API is an API that conforms to the design
principles of the REST,or representational state transfer architectural
style.For this reason, REST APIs are sometimes referred to RESTful APIs..
